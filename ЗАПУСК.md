# Инструкция по запуску SHIFT Messenger

## Подготовка

Убедитесь, что все зависимости установлены:
```bash
pip install -r requirements.txt
```

## Запуск сервера

Откройте **первый** терминал (командную строку) и выполните:

```bash
cd d:\Shift
python run_server.py
```

Или напрямую:
```bash
python server/server.py
```

Сервер запустится и покажет сообщение:
```
Запуск сервера SHIFT на localhost:8765
Сервер запущен и ожидает подключений...
```

**Не закрывайте этот терминал!** Сервер должен работать постоянно.

## Запуск клиента

Откройте **второй** терминал и выполните:

```bash
cd d:\Shift
python run_client.py
```

Или напрямую:
```bash
python client/gui.py
```

Откроется окно входа в мессенджер.

## Использование

### Первый запуск

1. В открывшемся окне нажмите кнопку **"Регистрация"**
2. Введите имя пользователя (например: `user1`)
3. Введите пароль (например: `123456`)
4. Нажмите **"Регистрация"**
5. Вы увидите сообщение об успешной регистрации
6. Теперь нажмите **"Войти"** с теми же данными

### Общение

1. После входа вы увидите главное окно
2. Слева - список пользователей (пока пустой)
3. Для тестирования запустите **второй клиент** в новом терминале:
   ```bash
   python run_client.py
   ```
4. Зарегистрируйте второго пользователя (`user2`)
5. После входа второго пользователя он появится в списке у первого
6. Выберите пользователя из списка и начните общение!

## Запуск нескольких клиентов

Для тестирования можно запустить сколько угодно клиентов:

```bash
python run_client.py
```

Каждый клиент должен быть запущен в отдельном терминале.

## Возможные проблемы

### "Нет подключения к серверу"
- Убедитесь, что сервер запущен (первый терминал)
- Проверьте, что сервер работает на `localhost:8765`

### "Пользователь уже существует"
- Выберите другое имя пользователя при регистрации

### Окно не открывается
- Убедитесь, что PyQt5 установлен: `pip install PyQt5`
- Попробуйте запустить тест: `python test_gui.py`

## Структура базы данных

База данных создается автоматически в папке `data/shift.db`

Таблицы:
- `users` - пользователи
- `messages` - сообщения
- `rooms` - групповые чаты (для будущего использования)
- `room_members` - участники комнат

## Остановка

Для остановки:
1. Закройте все окна клиентов
2. Нажмите `Ctrl+C` в терминале с сервером
