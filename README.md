# SHIFT Messenger

Мессенджер с открытым исходным кодом на Python с поддержкой реального времени.

## Возможности

- ✅ Регистрация и аутентификация пользователей
- ✅ Обмен сообщениями в реальном времени (WebSocket)
- ✅ История сообщений
- ✅ Список онлайн пользователей
- ✅ Графический интерфейс (PyQt5)
- ✅ Консольный интерфейс
- ✅ Хранение данных в SQLite

## Структура проекта

```
Shift/
├── server/
│   ├── __init__.py
│   └── server.py              # WebSocket сервер
├── client/
│   ├── __init__.py

│   ├── gui.py                 # Графический интерфейс
│   └── console_client.py      # Консольный интерфейс
├── data/

├── CONSOLE.bat                # Запуск консольного клиента
├── run_server.py              # Скрипт запуска сервера

├── README.md                  # Этот файл
└── ЗАПУСК.md                  # Подробная инструкция
```

## Быстрый запуск (Windows)

### Вариант 1: Консольный клиент (рекомендуется для тестирования)

1. **Запуск сервера**: Дважды кликните на `START.bat`
2. **Запуск клиента**: Дважды кликните на `CONSOLE.bat`

### Вариант 2: GUI клиент

1. **Запуск сервера**: Дважды кликните на `START.bat`
2. **Запуск клиента**: Дважды кликните на `CLIENT.bat`

### Вариант 3: Через командную строку

**Терминал 1 (сервер):**
```bash
python run_server.py
```

**Терминал 2 (клиент GUI):**
```bash
python run_client.py
```

**Или консольный клиент:**
```bash
python client/console_client.py
```

## Установка зависимостей

Если зависимости не установлены:
```bash
pip install -r requirements.txt
```

## Использование

### Консольный клиент

1. Запустите `CONSOLE.bat` или `python client/console_client.py`
2. Выберите "1. Регистрация" и создайте аккаунт
3. Выберите "2. Вход" для входа в систему
4. Выберите "1. Показать пользователей" для списка онлайн
5. Выберите "2. Выбрать чат" и введите имя пользователя
6. Выберите "3. Отправить сообщение" для общения

### GUI клиент

1. Запустите `CLIENT.bat` или `python run_client.py`
2. Нажмите "Регистрация" и создайте аккаунт
3. Войдите в систему
4. Выберите пользователя из списка слева
5. Начните общение!

## Запуск нескольких клиентов

Для тестирования чата запустите несколько клиентов:
- Дважды кликните на `CONSOLE.bat` или `CLIENT.bat` несколько раз
- Или запустите в разных терминалах

## Технологии

- **Python 3.8+**
- **websockets** - для обмена сообщениями в реальном времени
- **PyQt5** - для графического интерфейса
- **SQLite** - для хранения данных
