# История изменений SHIFT Messenger

Формат: для каждой версии перечислены заметные изменения для пользователя и разработчика.

---

## V 0.0.4

### Статусы пользователя

- Добавлен модуль `statuses.py`: фиксированный список из 13 статусов (в сети, готов поболтать, злой, дома, на работе, кушаю, отошел, не доступен, занят, не беспокоить, невидим, невидим для всех, не в сети).
- **Сервер**: хранение статуса для подключённых пользователей; сообщения `set_status` и рассылка `user_status`; в `get_users` — поля `statuses` и уточнённый список `online` (для статусов «не в сети», «невидим», «невидим для всех» пользователь не попадает в видимый онлайн, соединение WebSocket сохраняется); в ответе `connected` — `status` и `allowed_statuses`.
- **Клиент** (`client.py`): `set_status`, обработка `user_status`, поле `current_status` после входа.
- **GUI**: выпадающий список «Мой статус»; в списке контактов — строка вида `логин — статус`.
- **Консоль**: пункт меню «5. Установить статус»; в списке пользователей отображаются статус и пометка видимого онлайн.

### Непрочитанные сообщения

- Учёт по полю `is_read` в таблице `messages` (SQLite).
- **Сервер**: `mark_conversation_read`, `get_unread_counts`; после `get_history` — пометка переписки с выбранным собеседником прочитанной и отправка клиенту `unread_counts`; при доставке входящего `message` получателю — обновлённый `unread_counts`; в `get_users` — поле `unread_counts`; сообщение `mark_read` с полем `with_user`.
- **Клиент**: `mark_read`, событие `unread_counts`.
- **GUI**: счётчики непрочитанных у контактов и в заголовке панели; при открытом чате после входящего сообщения вызывается `mark_read`.
- **Консоль**: обработка `unread_counts`, вывод числа непрочитанных в списке пользователей.

### Окно чата и пузыри

- Область сообщений: **`QScrollArea` + виджеты** вместо одного `QTextEdit` с HTML — каждое сообщение в **`QFrame`** со стилем Qt (`border-radius`, фон, рамка) для стабильного отображения скругления.
- Ширина пузыря до **75%** области просмотра; **свои** сообщения — **справа**, **входящие** — **слева**.
- Внутри пузыря: **имя и время** одной строкой (слева), ниже — **текст** сообщения.
- При изменении размера окна пересчитывается максимальная ширина пузырей.

### Перенос длинных строк без пробелов

- Текст сообщения в **`QTextEdit`** (только чтение, без полос прокрутки): режимы **`WidgetWidth`** и **`WrapAnywhere`** — перенос по символам у длинных фрагментов без пробелов; высота подстраивается под содержимое; при ресайзе пересчитываются тела сообщений.

### Прочее

- В `README.md` обновлены возможности, структура проекта (`statuses.py`) и инструкции по GUI и консоли.
